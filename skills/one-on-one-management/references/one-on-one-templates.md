# One-on-One Meeting Templates

## Subordinate Outline Template

### Preparation (Complete before meeting)
- [ ] Performance indicators and metrics
- [ ] Key events since last meeting
- [ ] Current problems/concerns
- [ ] Future plans and needs
- [ ] Items for Hold File (non-urgent)

### During Meeting
- [ ] Present agenda items
- [ ] Discuss performance indicators
- [ ] Review events and problems
- [ ] Identify potential issues
- [ ] Note action items
- [ ] Schedule next meeting

## Hold File Template

### Non-Urgent Items to Batch
- [ ] Policy clarification needed
- [ ] Resource request (can wait)
- [ ] Process improvement suggestion
- [ ] Cross-functional coordination issue
- [ ] Training need identification

### Review Schedule
- Review hold file items at each one-on-one
- Move time-sensitive items to main agenda
- Archive resolved items

## Remote Meeting Checklist

### Before Meeting
- [ ] Receive subordinate's outline
- [ ] Review outline and prepare questions
- [ ] Test video/audio connection

### During Meeting
- [ ] Take detailed notes (compensate for missing visual cues)
- [ ] Confirm understanding frequently
- [ ] Explicitly state action items

### After Meeting
- [ ] Send summary notes
- [ ] Request confirmation of commitments
- [ ] Schedule next meeting
