# Visual Aids Guidelines

## Types of Visual Aids

### Tables
- Use for numerical data
- Keep simple and clear
- Highlight key figures
- Include units and labels

### Charts and Graphs
- Use for trends and comparisons
- Choose appropriate chart type
- Label axes clearly
- Use consistent colors

### Graphics and Diagrams
- Use for processes and relationships
- Keep uncluttered
- Use standard symbols
- Include legend if needed

## Design Principles

### Clarity
- Large, readable fonts (minimum 24pt)
- High contrast colors
- Simple layouts
- Minimal text per slide

### Consistency
- Same font throughout
- Same color scheme
- Same layout style
- Same terminology

### Emphasis
- Use color to highlight key points
- Use arrows or pointers
- Use bold or underline sparingly
- Animate only when necessary

## Timing Rule
- **4 minutes per visual aid**
- Includes presentation and discussion
- Plan accordingly for number of visuals
- Be prepared to skip if running behind

## Common Mistakes
- Too much text on one visual
- Too many colors
- Inconsistent formatting
- Too small to read
- Overly complex charts
- Reading from the visual
