# Operation Review Role Briefing Checklist

## For Organizing Manager

### Pre-Review Briefing for Presenters
- [ ] Explain the purpose and audience
- [ ] Clarify what issues to emphasize
- [ ] Specify level of detail required
- [ ] Review the 4-minute rule per visual aid
- [ ] Discuss time constraints and schedule
- [ ] Encourage preparation for questions
- [ ] Set expectations for professional delivery

### Pre-Review Briefing for Reviewing Manager
- [ ] Explain their role as catalyst and model
- [ ] Emphasize NO preview of materials
- [ ] Discuss importance of spontaneous reaction
- [ ] Encourage asking probing questions
- [ ] Remind them to set example for junior managers

### Pre-Review Briefing for Audience
- [ ] Explain their active participation role
- [ ] Emphasize that attendance is work, not passive
- [ ] Clarify responsibility to correct factual errors
- [ ] Set expectations for professional engagement
- [ ] Warn against disengagement behaviors

## Logistics Checklist
- [ ] Room reserved and configured
- [ ] Audiovisual equipment tested
- [ ] Backup equipment available
- [ ] Visual aids prepared and organized
- [ ] Agenda distributed
- [ ] Timekeeper assigned
- [ ] Evaluation forms (if used)

## During Review Monitoring
- [ ] Keep presentations on time (use inconspicuous gestures)
- [ ] Ensure audience participation
- [ ] Monitor presenter comfort and adjust if needed
- [ ] Track questions and answers
- [ ] Note key insights and decisions

## Post-Review Actions
- [ ] Collect feedback from presenters
- [ ] Debrief with reviewing manager
- [ ] Distribute minutes or summary
- [ ] Follow up on action items
- [ ] Plan next review if needed
