# Detailed Indicator Examples by Domain

## Manufacturing Indicators

**Paired Sets:**
- Production Volume + Defect Rate
- On-Time Delivery + Inventory Turns
- Equipment Uptime + Maintenance Cost

**Leading Indicators:**
- Raw material inventory levels
- Work in progress (WIP)
- Equipment temperature/pressure readings

**Lagging Indicators:**
- Units shipped
- Customer returns
- Scrap rate

## Administrative Indicators

**Paired Sets:**
- Vouchers processed + Error rate
- Calls answered + Abandonment rate
- Applications processed + Approval rate

**Leading Indicators:**
- Inbox backlog
- Pending requests
- Staff absenteeism

**Lagging Indicators:**
- Reports completed
- Customer satisfaction scores
- Turnaround time

## Sales Indicators

**Paired Sets:**
- Sales volume + Margin
- Calls made + Conversion rate
- Pipeline value + Deal velocity

**Leading Indicators:**
- New leads added
- Meetings scheduled
- Proposals sent

**Lagging Indicators:**
- Revenue booked
- Contracts signed
- Renewals completed
