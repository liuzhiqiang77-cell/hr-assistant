# The Sports Analogy Reference

## Context
This diagnostic approach is derived from the "Sports Analogy" section, which emphasizes that performance issues fundamentally stem from either capability or motivation.

## Key Insight
The top 5% of managers think deeply about their craft and distinguish between hands-on and hands-off approaches based on task-relevant maturity, while 95% do not.

## Interview Question
"Is it better to be a hands-on or hands-off manager?"

**Correct Answer**: It depends on the employee's task-relevant maturity. There is no universal answer.
