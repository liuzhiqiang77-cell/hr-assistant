## Paired Indicator Examples by Department

### Sales
- **Primary**: Revenue / Orders obtained
- **Counter**: Customer returns / Complaint rate / Margin %

### Production
- **Primary**: Units produced / On-time delivery %
- **Counter**: Defect rate / Rework % / Scrap cost

### Customer Service
- **Primary**: Tickets closed / Calls answered
- **Counter**: First-call resolution % / Customer satisfaction / Avg handle time

### HR / Recruiting
- **Primary**: Positions filled / Time to hire
- **Counter**: Retention rate / Quality of hire / Cost per hire

### Finance / Accounting
- **Primary**: Invoices processed / Reports completed
- **Counter**: Error rate / Days to close / Audit findings

### IT / Development
- **Primary**: Features released / Tickets resolved
- **Counter**: Bug rate / Uptime / System incidents

### Marketing
- **Primary**: Leads generated / Campaign reach
- **Counter**: Cost per lead / Conversion rate / Brand awareness
