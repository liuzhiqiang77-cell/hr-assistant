# Monitoring Examples

## Lowest-Added-Value Stage Examples

| Task Type | Low-Value Stage to Monitor | High-Value Stage to Avoid Monitoring |
|-----------|---------------------------|-------------------------------------|
| Report Writing | Rough draft outline | Final polished document |
| Software Development | Architecture/design review | Final QA testing |
| Project Planning | Initial work breakdown structure | Final presentation deck |

## TRM-Based Monitoring Schedule

| TRM Level | Monitoring Frequency | Sample Approach |
|-----------|---------------------|-----------------|
| Low (new task) | Daily or every 2-3 days | Check 30-40% of details |
| Medium (growing) | Weekly | Check 20-30% of details |
| High (experienced) | Bi-weekly or monthly | Check 10-20% of details |
