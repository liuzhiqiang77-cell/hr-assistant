# Output vs. Internal Measures by Role

## Sales Manager

### Output Measures
- Revenue achieved vs. quota
- Number of new accounts closed
- Average deal size
- Sales cycle length

### Internal Measures
- Team morale and retention
- Sales process improvements implemented
- Training and development of junior reps
- Cross-functional collaboration with marketing

**Weighting**: 70% Output / 30% Internal (revenue-focused quarter)

---

## Engineering Manager

### Output Measures
- Features delivered on schedule
- Bug count and severity
- System uptime
- Performance benchmarks met

### Internal Measures
- Code quality improvements
- Team technical growth
- Documentation quality
- Knowledge sharing initiatives

**Weighting**: 50% Output / 50% Internal (balanced focus)
