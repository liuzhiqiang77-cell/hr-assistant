# Delegation Checklist

## Before Delegating
- [ ] Shared operational values established
- [ ] Common information base exists
- [ ] Task is familiar to me (the manager)
- [ ] Subordinate's TRM assessed
- [ ] Clear expectations defined

## During Delegation
- [ ] Explain desired outcomes, not methods
- [ ] Provide necessary resources and authority
- [ ] Set checkpoints for monitoring
- [ ] Confirm understanding
- [ ] Document the delegation

## After Delegation
- [ ] Monitor at lowest-added-value stage
- [ ] Use sampling, not 100% inspection
- [ ] Adjust monitoring based on performance
- [ ] Provide feedback and coaching
- [ ] Retain accountability for results
