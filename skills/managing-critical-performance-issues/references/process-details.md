# Managing Critical Performance Issues - Detailed Reference

## Variable Definitions

- **current_stage** (enum): The current stage of the subordinate: Ignore, Deny, Blame Others, Assume Responsibility, Find Solution
- **evidence_available** (boolean): Whether the manager has sufficient facts and examples to demonstrate reality

## Result Template
Subordinate assumes responsibility and commits to a solution, OR subordinate follows direct instructions under position power with performance monitoring.

## Key Insight
The text emphasizes that 'Assuming Responsibility' is an emotional step (hard), while 'Finding Solution' is an intellectual step (easier).
