# Bonus Calculation Examples

## Example 1: Middle Manager - 3-Factor Split

### Compensation Profile
- **Base Salary**: $100,000
- **Bonus Target**: 20% of total compensation
- **Total Compensation**: $125,000
- **Bonus Pool**: $25,000

### Bonus Factors
| Factor | Weight | Target | Actual | Multiplier | Portion |
|--------|--------|--------|--------|------------|---------|
| Individual Performance | 50% | Exceeds | Exceeds | 1.2 | $15,000 |
| Team Objective | 30% | 100% | 110% | 1.1 | $8,250 |
| Corporate Financial | 20% | 10% growth | 8% growth | 0.8 | $4,000 |
| **Total Bonus** | | | | | **$27,250** |
