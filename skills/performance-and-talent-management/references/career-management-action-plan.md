# Career Management Action Plan

## Daily Practices

### Value Addition
- [ ] Identified one way to improve team output today
- [ ] Completed high-leverage task
- [ ] Documented contribution to organizational goals

### Connectivity
- [ ] Checked industry news/trends
- [ ] Connected with someone outside immediate team
- [ ] Shared relevant information with network
- [ ] Sought input on organizational direction

### Innovation
- [ ] Tried new tool or technique
- [ ] Experimented with different approach
- [ ] Learned new skill relevant to role
- [ ] Shared learning with team

## Weekly Review

### Performance Assessment
1. What value did I add this week?
2. What did I learn?
3. What connections did I make?
4. What innovations did I try?
5. What competitive advantages did I strengthen?

### Threat Analysis
1. Is my position blocking others?
2. Am I in a slow/no-growth environment?
3. Who are my global competitors?
4. What skills are becoming obsolete?
5. What new skills are in demand?

## Monthly Planning

### Skill Development
- Identify 2-3 skills to develop
- Create learning plan
- Allocate time for practice
- Measure progress

### Network Building
- Add 5-10 new connections
- Reconnect with existing contacts
- Share value with network
- Seek mentorship opportunities

### Market Awareness
- Research industry trends
- Identify emerging opportunities
- Monitor competitor activities
- Assess organizational health

## Quarterly Strategy

### Competitive Positioning
- Assess market value of current skills
- Identify skill gaps
- Plan skill acquisition
- Consider credential/certification needs

### Career Options
- Evaluate internal opportunities
- Research external market
- Identify potential employers
- Prepare for transitions if needed

### Personal Brand
- Update professional profiles
- Document achievements
- Collect testimonials/recommendations
- Create portfolio of work

## Annual Review

### Career Audit
- Am I adding real value?
- Am I plugged in?
- Am I innovating?
- Is my competitive advantage growing?
- Am I prepared for market changes?

### Strategic Decisions
- Should I stay in current role?
- Should I seek promotion?
- Should I change companies?
- Should I change industries?
- Should I start my own business?

### Long-Term Planning
- Set 3-5 year career goals
- Identify required skills and experiences
- Create development roadmap
- Establish milestones and metrics
