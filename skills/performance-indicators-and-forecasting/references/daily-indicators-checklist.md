# Daily Operational Indicators Checklist

## Morning Review Routine

### 1. Sales Forecast
- [ ] Review planned deliveries for today
- [ ] Note any special events or promotions
- [ ] Confirm capacity matches forecast

### 2. Variance Analysis
- [ ] Compare yesterday's actual vs. planned
- [ ] Identify root causes of variance
- [ ] Adjust today's forecast if needed

### 3. Raw Material Inventory
- [ ] Check stock levels of critical items
- [ ] IF too low: Order immediately
- [ ] IF too high: Cancel scheduled deliveries
- [ ] Document any quality issues

### 4. Equipment Condition
- [ ] Review breakdown log from yesterday
- [ ] Check maintenance schedule
- [ ] IF breakdown: Arrange repairs or reallocate line

### 5. Manpower Assessment
- [ ] Check staff availability
- [ ] Identify call-outs or absences
- [ ] IF understaffed: Call temp help OR reallocate staff

### 6. Quality Indicator
- [ ] Review customer complaint log
- [ ] Identify staff with excessive complaints
- [ ] IF complaints high: Speak to staff member immediately

## Escalation Rules

- Inventory < 1 day supply → Immediate order + notify manager
- Equipment down > 2 hours → Reroute production + notify maintenance
- Staff shortage > 20% → Call temporary help + notify HR
- Quality complaints > 5% → Stop line + investigate root cause
