# Detailed Indicator Design Examples

## Manufacturing Indicators

### Primary Pair
- **Output:** Units produced per day
- **Quality:** Defect rate / First-pass yield

### Secondary Pair
- **Efficiency:** Machine uptime percentage
- **Cost:** Scrap cost per unit

## Sales Indicators

### Primary Pair
- **Output:** Orders booked / Revenue
- **Quality:** Order accuracy rate / Customer satisfaction

### Secondary Pair
- **Activity:** Calls made / Meetings held
- **Conversion:** Lead-to-opportunity ratio

## Software Development Indicators

### Primary Pair
- **Output:** Features deployed / Story points completed
- **Quality:** Bug count / Test coverage

### Secondary Pair
- **Efficiency:** Cycle time
- **Cost:** Defect escape rate

## HR Indicators

### Primary Pair
- **Output:** Positions filled / Time to hire
- **Quality:** Retention rate / Performance ratings

### Secondary Pair
- **Activity:** Interviews conducted
- **Cost:** Cost per hire

## Customer Service Indicators

### Primary Pair
- **Output:** Tickets resolved / Calls handled
- **Quality:** First contact resolution / CSAT score

### Secondary Pair
- **Efficiency:** Average handle time
- **Cost:** Cost per ticket
