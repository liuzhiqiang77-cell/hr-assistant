# Linearity Indicator Examples

## Monthly Production Target: 1,000 Units

```
Units
1000 |                Target
     |               /|
 750 |            /   |
     |         /      |
 500 |      /         |  <-- Actual (Back-loaded)
     |   /            |
 250 | /______________|____
     |Week 1   2   3   4    Time
```

**Interpretation**: Production concentrated in week 4. Inefficient resource use - breakdown in week 4 could cause miss.

## Recruiting Campaign: 50 Hires by June

```
Hires
 50 |                Target
    |               /|
 40 |            /   |
    |          /     |
 30 |        /       |
    |      /         |  <-- Actual (Below Line)
 20 |    /           |
    |  /             |
 10 |/_______________|____
    |Jan Feb Mar Apr May Jun   Time
```

**Interpretation**: In April, only 15 hires (30%) but should be 33 hires (67%). Need 35 hires in 2 months = 17.5/month vs current 5/month rate.

# Stagger Chart Example

| Forecast Made | Jan | Feb | Mar | Apr | May | Jun |
|---------------|-----|-----|-----|-----|-----|-----|
| January       | 100 | 110 | 120 | 130 | 140 | 150 |
| February      |     | 105 | 115 | 125 | 135 | 145 |
| March         |     |     | 118 | 128 | 138 | 148 |
| April         |     |     |     | 125 | 135 | 145 |

**Analysis for June**:
- Jan forecast: 150
- Feb forecast: 145 (deteriorating outlook)
- Mar forecast: 148 (slight improvement)
- Apr forecast: 145 (stable)

**Trend**: Business outlook softened in February, stabilized since.

**Action**: Monitor closely, plan for ~145 units rather than original 150.
