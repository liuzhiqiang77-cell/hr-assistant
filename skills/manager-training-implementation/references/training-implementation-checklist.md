# Training Implementation Checklist

## Planning Phase
- [ ] Identified training needs through observation and subordinate input
- [ ] Prioritized training topics by urgency and impact
- [ ] Selected initial course topic (3-4 lectures)
- [ ] Identified available resources (materials, co-instructors)
- [ ] Set schedule with specific deadlines

## Development Phase
- [ ] Created course outline covering all key topics
- [ ] Developed first lecture content
- [ ] Identified knowledgeable subordinates for pilot delivery
- [ ] Prepared feedback forms

## Delivery Phase
- [ ] Delivered first lecture to pilot group
- [ ] Collected immediate feedback and questions
- [ ] Developed second lecture based on pilot feedback
- [ ] Delivered remaining lectures iteratively
- [ ] Documented common questions and answers

## Evaluation Phase
- [ ] Distributed anonymous feedback forms
- [ ] Reviewed feedback objectively (expecting mixed responses)
- [ ] Identified areas for improvement
- [ ] Assessed whether learning objectives were met
- [ ] Decided on scalability approach (repeat vs. train-the-trainer)

## Follow-Up Phase
- [ ] Observed subordinates applying new skills
- [ ] Provided coaching and reinforcement as needed
- [ ] Planned next training course based on remaining needs
- [ ] Celebrated successful application of new skills
