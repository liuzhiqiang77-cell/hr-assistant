# Indicator Design Examples

## Manufacturing Indicators

**Primary Pair**
- Units produced per hour
- Defect rate per 100 units

**Secondary Pair**
- On-time delivery rate
- Customer return rate

**Inventory Pair**
- Average inventory level (days of supply)
- Stockout incidents per month

## Administrative Indicators

**Accounts Payable**
- Vouchers processed per day
- Error rate (errors per 100 vouchers)
- Average processing time per voucher

**Human Resources**
- Positions filled per month
- Time to fill (days)
- New hire retention rate (90 days)

**Customer Service**
- Calls answered per hour
- Average call duration
- Customer satisfaction score
- First-call resolution rate
