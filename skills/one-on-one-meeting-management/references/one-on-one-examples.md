# One-on-One Meeting Examples

## Hold File Example Use Cases
- Budget questions that can wait 2 weeks
- Personnel issues not needing immediate resolution
- Process improvement ideas for future discussion

## Zinger Warning Signs
- Subordinate hesitates before meeting ends
- "There's one more thing..." at the 55-minute mark
- Sudden shift in body language or tone
- "I've been thinking about..." in final moments

## Meeting Outline Template Structure
1. Performance indicators and data
2. Events since last meeting
3. Current problems and concerns
4. Future plans and priorities
5. Items for hold file

## Observation Checklist During In-Person Meetings
- [ ] Workspace organization
- [ ] Document retrieval efficiency
- [ ] Interruption frequency
- [ ] Work approach and tools used
- [ ] Team interaction patterns
