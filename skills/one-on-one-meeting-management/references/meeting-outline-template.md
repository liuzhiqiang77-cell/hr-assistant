# One-on-One Meeting Outline Template

## Preparation Instructions for Subordinate

Create an outline covering:

### 1. Performance Indicators
- Key metrics since last meeting
- Any indicators signaling trouble

### 2. Recent Events
- Hiring/people issues
- Organizational problems
- Future plans

### 3. Potential Problems
- Concerns or intuitions
- Issues preoccupying you

### 4. Heart-to-Heart Topics
- Work satisfaction
- Frustrations or doubts
- Career concerns

## During Meeting
- Both parties take notes on this outline
- Add action items and commitments
- Move non-urgent items to Hold File
