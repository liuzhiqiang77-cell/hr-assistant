# Remote One-on-One Meeting Checklist

## Before Meeting
- [ ] Subordinate sends outline in advance
- [ ] Supervisor reviews outline and prepares questions
- [ ] Test video/audio connection
- [ ] Have note-taking document ready

## During Meeting
- [ ] Start with casual check-in (builds connection)
- [ ] Follow outline structure
- [ ] Apply "Ask one more question" principle
- [ ] Both parties take notes visibly
- [ ] Screen share when reviewing documents

## After Meeting
- [ ] Exchange notes via email
- [ ] Confirm action items and owners
- [ ] Schedule next meeting (rolling basis)
- [ ] Update hold file with non-urgent items
