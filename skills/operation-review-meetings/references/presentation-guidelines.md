# Operation Review Presentation Guidelines

## Visual Aid Best Practices

### Types of Visual Aids
- **Tables**: Data comparisons, trends, metrics
- **Charts**: Graphs showing relationships
- **Diagrams**: Process flows, organizational structures
- **Bullet points**: Key messages, summaries

### Design Principles
- **One idea per visual**: Don't overcrowd
- **Large text**: Minimum 24pt for readability
- **High contrast**: Dark text on light background
- **Simple graphics**: Avoid unnecessary decoration
- **Consistent format**: Same style throughout presentation

### Using Color for Emphasis
- **Primary color**: For main points or data
- **Secondary color**: For supporting information
- **Highlight color**: For critical items (use sparingly)
- **Red/yellow**: For problems or warnings
- **Green**: For positive results or achievements

## Audience Monitoring Techniques

### Facial Expressions to Watch
- **Nodding**: Agreement, understanding
- **Frowning**: Confusion, disagreement
- **Blank stare**: Disengagement, confusion
- **Looking around**: Boredom, distraction
- **Leaning forward**: Interest, engagement
- **Leaning back**: Relaxation, disengagement

### Body Language Indicators
- **Crossed arms**: Defensiveness, resistance
- **Open posture**: Receptiveness, engagement
- **Note-taking**: Interest, desire to remember
- **Checking watch/watch**: Boredom, time pressure
- **Whispering to neighbor**: Confusion, disagreement

### Adjustment Strategies

**If audience looks confused:**
- Slow down
- Rephrase the point
- Provide an example
- Ask if clarification is needed

**If audience looks bored:**
- Speed up slightly
- Move to next point
- Ask a question to engage
- Use a more emphatic tone

**If audience is distracted:**
- Pause briefly
- Make eye contact
- Use a pointer to redirect attention
- Ask a direct question

## Time Management Strategies

### Before the Meeting
- **Rehearse**: Time your presentation
- **Prioritize**: Know what to cut if short on time
- **Prepare transitions**: Smooth flow between visuals

### During the Meeting
- **Watch the clock**: Glance at time periodically
- **Use signals**: Organizing manager may gesture to speed up
- **Be flexible**: Adjust depth based on remaining time
- **Skip ahead**: If running late, jump to conclusions

### Common Time Problems

**Running too fast:**
- Add examples
- Elaborate on key points
- Allow more questions

**Running too slow:**
- Skip secondary points
- Combine visuals
- Move questions to end
- Provide handouts for details

## Question Handling

### Types of Questions
- **Clarification**: "Can you explain that number?"
- **Challenge": "Why did you choose that approach?"
- **Suggestion": "Have you considered X?"
- **Expansion": "What are the implications?"

### Response Strategies
- **Listen fully**: Don't interrupt
- **Acknowledge**: "That's a good question"
- **Answer directly**: Address the core issue
- **Be honest**: Admit if you don't know
- **Offer follow-up**: "I'll get you that data"

### Handling Challenges
- **Stay calm**: Don't get defensive
- **Provide evidence**: Use data to support your position
- **Acknowledge validity**: "That's a valid concern"
- **Explain reasoning**: Show your thought process
- **Seek input**: "What would you suggest?"
