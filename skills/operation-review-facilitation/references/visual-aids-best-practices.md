# Visual Aids Best Practices for Operation Reviews

## General Guidelines

### Content Design
- One main idea per visual
- Use clear, readable fonts (minimum 24pt for presentations)
- Limit text: use visuals, not documents
- Use color to highlight emphasis, not decoration

### Types of Visual Aids

**Tables:**
- Use for comparative data
- Highlight key rows/columns with color
- Keep headers clear

**Charts/Graphs:**
- Choose the right type for your data
- Label axes clearly
- Show trends, not just raw numbers

**Graphics:**
- Use diagrams to show processes or relationships
- Ensure they're large enough to see
- Use consistent style throughout
