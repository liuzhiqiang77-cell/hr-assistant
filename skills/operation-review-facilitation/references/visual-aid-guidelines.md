# Visual Aid Guidelines for Operation Reviews

## Types of Visual Aids
- Overhead transparencies
- Data tables
- Numerical charts
- Graphics and diagrams

## Time Allocation
- **4 minutes per visual aid** (presentation + discussion)
- Includes time for questions and clarifications

## Presentation Best Practices

### Engagement
- Engage eyes and ears simultaneously
- Use color to highlight emphasis
- Use pointer to direct attention

### Audience Monitoring
- Watch facial expressions for confusion
- Monitor body language for disengagement
- Adjust delivery speed based on feedback
- Clarify points when signs of misunderstanding appear

### Common Mistakes to Avoid
- Too much text on one visual
- Reading from the visual instead of speaking
- Moving too quickly through complex data
- Failing to highlight key points
- Ignoring audience cues
